# Kid Profiles

Each file here describes one child's school, academic level, interests, and activities.
Skills like `/weekend-fun-events`, `/weekly-digest`, `/youtube-kids-audit`, and
`/find-school-calendar` read these files to personalize their output.

## How to create a profile

Copy `_template.md` to `<child-name>_profile.md` and fill it in.

Or run `/youtube-kids-audit analyze-kids-content` — if no profile exists for your child,
the skill will ask you the questions and write the file automatically.
