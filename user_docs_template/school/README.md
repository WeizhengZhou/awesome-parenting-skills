# School Files

Auto-populated by skills. You don't need to create these manually.

```
school/
├── calendar/          ← created by /find-school-calendar
│   └── [school-slug]-[YYYY-YY].md
└── newsletter/        ← created by /agentmail-skill scan-emails
    └── [YYYYMmmDD].md
```

## calendar/

One file per school per year. Created by running:
```
/find-school-calendar
```

## newsletter/

One file per newsletter. Created when you forward school emails to your AgentMail inbox
and run `/agentmail-skill scan-emails`.
