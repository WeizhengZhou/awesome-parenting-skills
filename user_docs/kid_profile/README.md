# Kid Profiles

Each file here describes one child's cognitive style, academic level, and interests.
The `analyze-kids-content` sub-skill reads these files to filter watch history and generate
a personalized educational analysis.

## How to create a profile

Copy `_template.md` to `<child-name>.md` and fill it in.
Or run `/youtube-kids-audit analyze-kids-content` — if no profile exists for your child,
the skill will ask you the questions and write the file automatically.
